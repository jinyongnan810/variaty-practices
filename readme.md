# 🎯 Variaty Practices

Hands-on practice repository covering backend APIs, frontend apps, data work, scripting, and tooling experiments.

## 📚 Projects

### Backend / API

- [django_basics](django_basics): Django + DRF fundamentals, API patterns, pytest-based testing and mocking.
- [cc-sdd-practices](cc-sdd-practices): Personal wiki API built with Django 6 + DRF + JWT auth, tags, and keyword search, implemented with a spec-driven workflow.
- [flask-practices](flask-practices): Flask basics for routes, templates, JSON responses, and simple locking patterns.
- [docker-uwsgi-flask-practices](docker-uwsgi-flask-practices): Production-style Flask deployment with uWSGI, Nginx, Docker Compose, and tests.

### Frontend / Full-Stack

- [nextjs-practices](nextjs-practices): Next.js 15 + React 19 + TypeScript + Tailwind 4, with Jest and Playwright setup.
- [web-tricks](web-tricks): React + TypeScript + Vite gallery of practical web UI/UX tricks, including migrated legacy demos and live detail pages.

### Data / Analytics

- [pandas-practices](pandas-practices): Data manipulation and analysis with pandas.
- [polars-practices](polars-practices): High-performance dataframe workflows with Polars.
- [graph-practices](graph-practices): Visualization-focused notebook examples using matplotlib.

### Tooling / Infra / Automation

- [dotenvx-practice](dotenvx-practice): Encrypted multi-environment config management with dotenvx.
- [mcp-practices](mcp-practices): MCP server configuration patterns (Supabase, GitHub, AWS docs, Apple docs, filesystem).
- [shell-practices](shell-practices): Bash automation examples for env export and batch rename workflows.

### Documentation / Validation

- [plantuml-practices](plantuml-practices): PlantUML sequence diagram practice.
- [json-schema-practices](json-schema-practices): JSON Schema validation examples in Python.

### Python Language Patterns

- [python-practices](python-practices): Concurrency and locking examples (threading, thread pools, fasteners) plus Pydantic practice.

## 🚀 Quick Start

Each project has its own README with exact steps. Common commands:

### Python projects

```bash
poetry install
eval $(poetry env activate)
```

or

```bash
pip install -r requirements.txt
```

### Node.js projects

```bash
npm install
npm run dev
```

### Notebook projects

```bash
jupyter notebook
```
