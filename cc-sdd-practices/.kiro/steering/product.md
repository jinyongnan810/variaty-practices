# Product Overview

**wikinn** is a personal wiki application — "Kinn's WiKi" — for organizing and sharing knowledge.

## Core Capabilities

- Knowledge organization and retrieval via a wiki-style interface
- RESTful API for programmatic access to wiki content
- Admin interface for content management

## Target Use Cases

- Personal knowledge base for capturing and organizing information
- API-driven content access for integrations or frontends
- Practice project for Spec-Driven Development (SDD) workflows

## Value Proposition

A lightweight, self-hosted wiki built on Django with a REST API layer, developed using a structured AI-assisted development lifecycle (AI-DLC with Kiro-style SDD).
